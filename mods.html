Mods

¿Que son Mods?

Es una extensión del software que modifica un videojuego original proporcionando nuevas posibilidades, ambientaciones, personajes, diálogos, objetos, etc.

-Recomendados

Skse

Parche para el juego el cual permite la instalacion y el funcionamiento correcto de muchos mods

SkyUI

Mod que permite cambiar la interfaz de usuario a una con mejor apariencia y comodidad.

Unofficial Skyrim Legendary Edition Patch

Parche el cual corrige muchos errores o bugs que vienen por defecto en el juego.

CTD and Memory patch ENBoost

Parche optimizado x64 para juegos muy modificados, utiliza la memoria de manera mucho más eficiente. También aumenta el rendimiento del juego.
